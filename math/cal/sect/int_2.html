Definite Integrals

A definite integral is used to find the area under or between a function and the x-axis. It is definite because it is restricted to a closed interval and the function must be continuous within that interval.

First Fundamental Theorem of Calculus

Basically, it says that the area under the function f (x) on interval [b, a], where F is the antiderivative of f, can be modeled using the following:
      ∫ab  f (x) dx = F(a) − F(b)
An example could be ∫32  4x3 dx which, in interval notation, would be x4 |32. This simplifies to 34 − 24 which is 81 − 16 which equals 65.

A special case of definite integrals are absolute value composite functions such as |x2 − 4|.
Say we have g(x) =  ∫ab  |f (x)| dx. The absolute value function is actually a piecewise function made up of two components: a negative and a positive. The negative component is for when f (x) < 0 and the positive component is for when f (x) > 0. Therefore, we rewrite g as the following if f is negative when x is less than c:
      g(x) = ∫cb  −f (x) dx + ∫ac  f (x) dx
