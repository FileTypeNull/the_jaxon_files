Power Series

A series that is defined by a function is a power series. For example, take the following series:

      g(x) = Σ n = 1 ∞ x⁄n

In this case, g(x) will have a certain interval of convergence, or a certain set of x-values that will allow for the series to converge. For most power series, the best method is the ratio test to find this interval of convergence.

Taylor Polynomials

A Taylor polynomial is a polynomial that represents some function. For example, you can represent the function ex with the following power series:

      ex = Σ n = 0 ∞ x⁄n!

The n-th term in a Taylor polynomial for the function f (x) should follow this structure: f (n)(c) ⁄n! × (x − c)n. Breaking down this structure, we know:
  • f (n)(c), which is the value of the n-th derivative of f (x) at x = c
  • The value c, which is the center for the function that we will use to create our Taylor polynomial; when c = 0, it is known as a Maclaurin polynomial

How can we find these values? The best way is to make a table with the following format:
n f (n)(x) f (n)(c) f (n)(c) ⁄n!
0
1
⋮

Say we are given the function f(x) = xx. Find the third degree Taylor polynomial for the function centered at x = 1:
n f (n)(x) f (n)(1) f (n)(1) ⁄n!
0 xx 1 1⁄0! = 1
1 xx(ln x + 1) 1 1⁄1! = 1
2 xx(ln x + 1)2 + x x−1 2 2⁄2! = 1
3 x x−2(x(ln x + 1)(x(ln x + 1)2 + 3) − 1) 3 3⁄3! = ½
