Logarithmic Differentiation

If the product rule, quotient rule, and chain rule were too difficult for you, then logarithmic differentiation is your key to success.

When finding the derivative of the function f(x) = tan²x(3x + 2), it might be more practical to find the natural logarithm of both sides of the function:   ln f(x) = ln( tan²x(3x + 2) ).
Using properties of logarithmic functions, this can be simplied to:   ln f(x) = 2ln(tan x) + ln(3x + 2).
Now what's next?

Natural Log and e to the x

Because of the following proof (which you do not need to know), the derivative of y = ln f(x) is  f '(x)⁄f(x)
  → y = ln f(x)
  → e y = f(x)
  → dy⁄dx × e y = f '(x)
  → dy⁄dx = f '(x)⁄e y
  → dy⁄dx = f '(x)⁄f(x)

Knowing this, we can determine the derivative of the function we had differentiated:
  → ln f(x) = 2ln(tan x) + ln(3x + 2)
  → f '(x)⁄f(x) = 2(sec²x⁄tan x) + 3⁄3x + 2
  → f '(x) = f(x)×(2csc x sec x + 3⁄3x + 2)
  → f '(x) = tan²x(3x + 2)×(2csc x sec x + 3⁄3x + 2)
  → f '(x) = (6x + 4)sec²xtan x + 3tan²x
You can check your work using the product and chain rules.

Say we have the function y = e f (x). What is the derivative of this function?
Using logarithmic differentiation, the function becomes ln y = f (x). This therefore becomes y'⁄y = f '(x) which simplifies to y' = e f (x) × f '(x).

If it were g(x) = ee(2x3 + 3x), then what would be g'(x)?
  → g'(x) = ee(2x3 + 3x) × d(e(2x3 + 3x))⁄dx
  → g'(x) = ee(2x3 + 3x) × e(2x3 + 3x) × d(2x3 + 3x)⁄dx
  → g'(x) = ee(2x3 + 3x) × e(2x3 + 3x) × (6x2 + 3)
  → g'(x) = (6x2 + 3)e(e(2x3 + 3x) + 2x3 + 3x)

Applications of Logarithms

How would you find the derivative of the function y = xx ? Well, you would differeniate it through the use of natural logarithm ( ln ). How?
First, take the natural log of both sides: ln y = ln xx
Using the properties of logarithms, the exponent x can be brought ouside the natural log: ln y = xln x
Using what we know about the derivative of natural logarithms, we can deduce the result for when we take the derivatives of both sides of the function: y '⁄y = (xln x)'
To evaluate the right side, we need to use product rule: y ' = y(ln x + x × 1⁄x)
Finally, simplify and replace y with the value of the function: y ' = xx(ln x + 1)

If you want to find the derivative of, say, k(x) = πlog7(8x − 3), we can use the same strategy.
The first step gives us ln k(x) = log7(8x − 3) × ln π . And now we've reached a barrier. What do we do with a base-7 logarithm?
Well, any logarithm can be rewritten using the natural logarithm as such:   logba = ln a⁄ln b . Therefore, we can rewrite the function as: ln k(x) = ln (8x − 3) ÷ ln 7 × ln π
Let's make this look a bit prettier by combining the two constant natural logs (ln 7 and ln π) into one logarithm: ln k(x) = ln (8x − 3) × log7π
Take the derivative of both sides: k '(x) = k(x) × 8⁄8x − 3 × log7π
Finally, simplify: k '(x) = πlog7(8x − 3) × 8⁄8x − 3 × log7π

Implicit Differentiation

Imagine having to find the tangent line to a point on a circle, given the function for that circle. How would you do that?
Well, say we have the following circle and we want to find the tangent line to the point ( 3, 2 ): x² + y² = 13

To differentiate this function, all we need to do is treat y like x, but we need t use chain rule because y is actually a function instead of a variable. Therefore, if we differentiate the circle, we end up with:
    2y × dy⁄dx + 2x = 0
Now, to find the value of dy⁄dx, we just need to isolate the derivative:
    dy⁄dx = − x⁄y
Now plug in the coordinates of our point into the point-slope form for a tangent line:
    y − 2 = 3⁄2(3 − x)

L'Hopital's Rule and Indeterminates

Remember that a limit can yield an indertminate value, such as 0 ÷ 0 or ∞ ± ∞. When you have a limit that has a numerator and denominator and results in an indeterminate value, you can differentiate the numerator and denominator separately as two different functions until you achieve a solution. This is known as L'Hopital's rule.

Here are some examples of L'Hopital's rule in use with the indeterminate forms 0 ÷ 0 and ∞ ÷ ∞:
  1. lim x → 0 (sin x⁄x) = 0 ÷ 0 → lim x → 0 ((sin x)'⁄(x)' = cos x⁄1) = 1
  2. lim x → ∞ (5x − 8⁄3x + 1) = ∞ ÷ ∞ → lim x → ∞ ((5x − 8)'⁄(3x + 1)' = 5⁄3) = 5⁄3

It is harder to use L'Hopital's rule for indeterminate values that are not immediately in a form where there is a division of two functions. The goal is to try and turn the limit into a division of two functions and then use L'Hopital's rule.
