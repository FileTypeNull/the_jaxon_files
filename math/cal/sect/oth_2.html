In algebra, we learned that a sequence is a set of numbers that follow a set order. There are two types: arithmetic and geometric. An arithmetic sequence is a set of numbers of which the difference between consecutive terms is constant. A geometric sequence is a set of numbers of which the ratio between consecutive terms is constant. A series is the sum of all the terms in some defined sequence. It is represented using 'Σ' for summation.

Geometric Series

In a geometric sequence, we have the expression an = r × an − 1, where r is the ratio and an − 1 is the previous term in the sequence.
In a geometric series, we implement the summaton notation Σ. This makes our expression:

      s = Σ n = 0 ∞ a0(r)n

Now, a series can either converge or diverge. A convergent series is one that approaches a specific value, and a divergent series is one that does not. If a geometric series has a ratio that is constrained within the bounds of (-1, 1), then it will converge towards the value of a0(1 − r)-1. That means that if the initial value is 1 and the ratio is ½, then the series will converge to 2.

p-Series

A p-series is one that follows the following structure:

      s = Σ n = 1 ∞ k⁄n p

If p is greater than 1, then the series will converge. If p is less than or equal to 1, then the series is divergent. A notable example of this type of series is the Basel problem (here is a child of this problem):

      s = Σ n = 1 ∞ 6⁄n2 = π2

n-th Term

The n-th term test is a way to test if a series will certainly diverge; however, it has some flaws in that it cannot determine if a series will certainly converge.
How does it work? Say we have the following series:

      s = Σ n = 0 ∞ an

The n-term test for this series would be finding limn → ∞ an . If the limit does not approach zero, then the series definitely diverges. If the limit does approach zero, then further tests must be done in order to determine the convergence or divergence of the series.
The flaw of this test can be seen in the harmonic series:

      s = Σ n = 1 ∞ 1⁄n

The harmonic series is divergent, but the n-th term test determines that the limit does in fact approach zero. Therefore, it cannot accurately determine if the series is convergent and is only useful for enabling future testing methods, such as the integral test, or declaring that a series definitely diverges. Furthermore, any convergent series must pass the n-th term test in order for that series to be convergent. In other words, all convergent series pass the n-th term test but passing the n-th term test does not make a series convergent.

Telescoping Method
This requires the ability to rewrite a certain series in the foollowing way:

      s = Σ n = a ∞ (an − bn)  =  Σ n = a ∞ an − Σ n = b ∞ an

For example:

      s = Σ n = 1 ∞ (1⁄n − 1⁄n + 1)  =  Σ n = 1 ∞ 1⁄n − Σ n = 2 ∞ 1⁄n  =  Σ n = 1 1 1⁄n = 1

Note that this only works if the series passes the n-term test. This is because if you write out the complete expanded form of the series, you would get end up needing to find the n-th term to solve for the series.

Integral Test
In order to perform the integral test, there are three constraints for the series:
  • limn → ∞ an = 0
  • an decreasing
  • an > 0


If the series passes all three of these constraints, then it is possible to determine the convergence of the series by means of the integral test. The following is the formula for the integral test:

      s = Σ n = a ∞ an  →  an = f (n)  →    ∫∞a  f (x) dx

If the integral results in a number, the series will converge (but not to that number). If the integral diverges, so will the series.

Comparison Test(s)
Say you have a function that you'd rather not integrate for time's sake, but it looks rather similar to another series you've seen before. Well, in this case, you can use the direct comparison test. For example:

  sgiven = Σ n = 1 ∞ (n2 + 1)-1

This series is very similar to the following known series:

  sknown = Σ n = 1 ∞ 1⁄n2

Now, we can compare them like this:

  Σ n = 1 ∞ (n2 + 1)-1  <  Σ n = 1 ∞ 1⁄n2

Since sknown is convergent, then sgiven converges because it is less than some number. However, it only absolutely converges if the absolute value of the series also converges. If the absolute value diverges, then the series only conditionally converges. If the known series is divergent and the given series is greater than that known series, then the series diverges. If neither of these constraints are met, then there must be further testing. This can be done using the limit comparison test, the formula for which is the following:
      limn → ∞ | sgiven ÷ sknown | = L
If L is a number, then the given series will follow the same behavior as the known series (i.e. if the known series diverges, the given series will also diverge). If L is infinity or zero, then the given series' behavior is impossible to determine with the limit comparison test.

Ratio and Root Tests

Ratio Test
  • limn → ∞ | an + 1 ÷ an | = L
  • if L < 1, the series converges
  • if L > 1, the series diverges
  • if L = 1, it is impossible to determine the convergence/divergence of the series

Root Test
  • limn → ∞ | an |1⁄n = L
  • if L < 1, the series converges
  • if L > 1, the series diverges
  • if L = 1, it is impossible to determine the convergence/divergence of the series
