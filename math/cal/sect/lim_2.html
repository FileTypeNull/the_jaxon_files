The Formal Definition of a Limit

Let's refer to the informal definition of a limit: limx → c f(x) = L.
This definition is informal because exact meanings have not yet been given to the two phrases:
  "f(x) becomes arbitrarily close to L"
and
  "x approaches c"

The first person to assign mathematically rigorous meanings to these two phrases was Augustin-Louis Cauchy. His ε-δ definition of limit is the standard used today.

Let ε represent a small positive number (such as 0.01). The phrase "f(x) becomes arbitrarily close to L" means that f(x) ∈ (L − ε, L + ε). Using absolute value, this can be written as
  | f(x) − L | < ε
Similarly, the phrase "x approaches c" means that there exists a positive number δ such that x ∈ (c − δ, c)∪(c, c + δ). This fact can be expressed by the double inequality
  0 < | x − c | < δ

Using the epsilon-delta definition, what is a reasonable estimate for lima → 2 m(a)?
First, determine the δ with 0 < | a − c | < δ. This means that 0 < | a − 2 | < δ; therefore, (a − 2) ∈ (-δ, δ) which becomes a ∈ (2 − δ, δ + 2).

Next, determine what values of a should be used to determine the limit. Start with δ = 0.1, and then solve for m(a) where a ∈ (2 − δ, δ + 2) to determine the δ-intervals. Then divide δ by 10 repeatedly to get a more precise interval with each division.

For m(a), what is ε when δ = 0.01? First, determine m(c ± δ) {3.96, 4.04}. Then, find the absolute value of the difference between those two results. This yields 2ε = 0.08, so divide by 2. Therefore, ε = 0.04.

Finally, the limit is within reach. If | m(a) − L | < ε, then | 3.96∪4.04 − L | < 0.04. Therefore, L = 4.



Trigonometric Limits

There are two defined limits that are important to remember when dealing with the limits of trigonometric functions:
  • limx → 0 (1⁄xsin x) = 1
  • limx → 0 (1⁄x(1 − cos x)) = 0

What is limx → 0 [(cos2x − 1) ÷ (cos x − 1)]?
When we insert x = 0 into the equation, the result is 0⁄0. This result is indeterminate, meaning there is a hole at x = 0 and an existing limit. "0 over 0, keep going."
The first step is to use the double angle theorem (see Precalculus) to separate cos2x into a factorable component. In this case, we want to use cos2x = 2cos²x − 1 because we want to have one trigonmetric functon throughout the entire problem.
Then, we substitute and simplify: limx → 0 [(2cos²x − 2) ÷ (cos x − 1)].
Notice that the numerator can be factored twice: by undistributing the 2 and using difference of squares. Thus, limx → 0 [(2(cos²x − 1)) ÷ (cos x − 1)] = limx → 0 [(2(cos x − 1)(cos x + 1)) ÷ (cos x − 1)].
The factor (cos x − 1) is in both the numerator and denominator, so it can be removed from the problem entirely: limx → 0 [2(cos x + 1)].
Finally, we can input x = 0 into the equation to find the limit: limx → 0 [2(cos(0) + 1)] = limx → 0 [2(1 + 1)] = limx → 0 4.
Referring back to our properties of limits, the limit of a constant is the same as the value of that constant. Therefore, the limit is 4.



Behavior of Functions Using Limits

Look at the graph d(x). Its end behavior can be defined as the following:
  • limx → -∞ = 0
  • limx → 0 = unbounded
  • limx → ∞ = 0

Infinities and Zeroes
When determining the end behavior of a function without graphing, it is useful to know how numbers, zeroes, and infinities operate mathematically. Below is a list of properties of infinity and zero, along with their relation with the number n.
  • n ÷ ∞ = 0
  • ∞ ×/÷ n = ∞
  • ∞ ÷ ∞ = indet.
  • 0 ×/÷ n = 0
  • n ÷ 0 = DNE
  • 0 ÷ 0 = indet.
  • ∞ ± n = ∞
  • ∞ ± ∞ = indet.
  • n − ∞ = -∞
  • n ÷ n = 1
Using those properties of infinity and zero, we can determine the end behavior of limits without graphing. If the graph of the function were unkown, but we were given the equation f(x) = x²⁄x² + 2, then what is limx → ∞ f(x)?
In this case, we factor out the largest degree from both the numerator and denominator: f(x) = x²⁄x²(1 + 2⁄x²). The 2⁄x² becomes 0 because x = ∞. The x² can be cancelled out. Therefore, the final result is f(∞) = 1⁄1, or 1.
Conclusion: limx → ∞ f(x) = 1.

If f(x) = - ;x²⁄x − 2, what is limx → -∞ f(x)?
Start by factoring out the largest degree from both the numerator and denominator: f(x) = x²(-1)⁄x(1 − 2⁄x). The (− 2⁄x) simplifies to 0 because x = -∞. The x² and x cancel and leave x on the numerator. Thus, the solution to f(-∞) = -(-∞)⁄1 = ∞.
Conclusion: limx → -∞ f(x) = ∞.

If f(x) = √(x² + 2x + 1), what is the value of limx → -∞ f(x)?
First, factor the largest degree: f(x) = √(x²(1 + 2⁄x + 1⁄x²)). Then, we cancel our values using the divide by infinity rule: f(x) = √(x²). Now, we simplify. However, because our x-value is less than 0, it becomes f(x) = -x, because a negative times a negative is a positive. Finally, input the negative infinity: f(-∞) = -(-∞) = ∞.
Conclusion: limx → -∞ f(x) = ∞.

The Speed of a Function
When one function is "faster" than another, its range reaches infinity more quickly. This is because there are different sizes of infinity. In order from the slowest to fastest functions, it is: logarithmic, linear, power (xk), exponential, and factorial.

When a function g(x) is comprised of a slow divided by a fast, then limx → ∞ g(x) = 0. Because the denominator reaches infinity before the numerator, we call back to our rule that a number divided by infinity is equal to 0. Continuity
When a function is continuous, every x-value has a defined y-value and does not have any holes, asymptotes, or jumps—a jump is where the x-value has both an undefined y-value and a defined point.

How does continuity relate to limits?

There are three conditions a function must follow in order to be considered continuous:
  1. f(x) is a function that has a domain of (-∞, ∞)
  2. There is a bidirectional limit for every value of x
  3. limx → c f(x) = f(c) for any value of c
There are two types of dicontinuities: removable and unremovable. Holes are removable, while asymptotes and jumps are unremovable. This is beacause holes can still be the limit of the function, but not the defined point at that x-value.

When describing the continuity of a function, it is best to write the domain without including points of discontinuity.
Consider the following function: h(k).
When describing the continuity of this function within the interval [-4, 4], we would say that kcont ∈ (-4, -2)∪(-2, 0)∪(0, 2)∪(2, 4). Each break in the continuity of h is a jump.

Here's another example: g(z).
There's a jump discontinuity at z = 2, so the continuity of g is zcont ∈ (-∞, 2)∪(2, ∞).
