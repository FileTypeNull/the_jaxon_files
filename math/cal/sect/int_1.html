Indefinite Integrals

What is an indefinite integral? An indefinite integral is a type of process that results in a function's anti-derivative. But, what is an anti-derivative? Imagine that you have a function f (x). The anti-derivative of that function F(x) exists such that the derivative of the anti-derivative F '(x) is the same as the original function, F '(x) = f (x).

The symbol for integral is: ∫

Basic Integration Rules
  • ∫ xn dx = xn + 1 ÷ (n + 1) + C, n ≠ -1
  • ∫ dx = x + C  
  • ∫ k dx = k ∫ dx = kx + C  
  • ∫ k f(x) dx = k ∫ f(x) dx = kF(x) + C  
  • ∫ 1⁄x dx = ln |x| + C
  • ∫ [f (x) ± g(x)] dx = ∫ f(x) dx ± ∫ g(x) dx = F(x) ± G(x) + C
  • ∫ ex dx = ex + C
  • ∫ cos x dx = sin x + C  
  • ∫ sin x dx = -cos x + C  
  • ∫ sec²x dx = tan x + C  
  • ∫ csc²x dx = -cot x + C  
  • ∫ sec x tan x dx = sec x + C  
  • ∫ csc x cot x dx = -csc x + C  
  • ∫ 1⁄√(1 − x²) dx = arcsin x + C
  • ∫ 1⁄x² + 1 dx = arctan x + C
  • ∫ 1⁄x√(x² − 1) dx = arcsec |x| + C
The reason why each of these integrals has a C appended to the end is because the derivative of a constant C is 0, so there are infinite many solutions to an indefinite integral.

Substitution Method

Sometimes, the function within the indefinite integral can be too complex to integrate using the basic rules of integration.
Say you are trying to find
    ∫ x cos(x2) dx
Where would you start? Well, it's easier if we replace the inside function with some variable. In this case, we'll use u. So, if we set u = x2, then what do we do now? Well, we need to replace dx, which represents the derivative of the variable x, with du. How? We find the derivative of our u substitute: du = (x2)' = 2x dx. We maintain the dx in our du because x could be a representation of some unkown function.
In our original integral, we substitute in the u:
     ∫ x cos u dx
Now, we have a problem with the fact that we still have dx in our integral. This can be fixed by substituting in our du for dx. Therefore, we must find the value of dx in terms of du: du = 2x dx → dx = du⁄2x. Now, we substitute:
     ∫ x cos u × du⁄2x
Finally, we can simplify and apply our basic integration rules:
     ½ ∫ x⁄x cos u du   →   ½ ∫ cos u du   →   ½ × sin u + C   →   ½sin(x2) + C

The underlying concept in substitution is knowning what to substitute. This is the hardest part of the substitution method. If you don't know what to substitute, you will have a hard time solving the problem.
For instance, what would you substitute in the following integral:
     ∫ 1⁄x ln²x dx ?
It's like a puzzle. If you understandwhat you should be looking for, then you how to solve it. Since 1⁄x is the derivative of ln x , then we should use ln x as our substitute.
Set u = ln x , and therefore du = dx⁄x. Since 1⁄x is already in our integral, then we can substitute 1⁄x dx with our du:
     ∫ u2 du   →   u3 ÷ 3 + C   →   ⅓ln³x + C

Integration with Inverse Trigonometry

The inverse trig functions are already defined in the basic integration rules, right? Yes, but not every inverse trig integral will be so simple. It is important that you recognize when the function inside the integral is the derivative of an inverse trig function.
The result can only be an inverse trig function if the degree of the numerator is one less than half the degree of the denominator (not counting the radical)!

Consider the following integral:
     ∫ k⁄x² + b² dx
This follows the same structure as the integral function that yields arctan x . We can follow our basic integration rules and move k outside of the integral, giving us k ∫ f (x) dx . Now, we need to get "+ 1" in the denominator. This can be done if we remove the whole b2 value outside of the integral:
     k⁄b² ∫ dx⁄(x²⁄b² + 1)
Since the numerator and denominator of x²⁄b² is squared, we can bring out the power of two, resulting in (x⁄b)2. Now, let's set u = x⁄b for our substitution method. This makes du = 1⁄b dx , so dx = b du. Therefore, our integral is now:
     k⁄b² ∫ b × du⁄u² + 1   →   k⁄b² × b ∫ du⁄u² + 1   →   k⁄b ∫ du⁄u² + 1   →   k⁄b arctan u + C   →   k⁄b arctan (x⁄b) + C

Let's do the same thing, but with the following integral:
     ∫ k⁄√(b² − x²) dx
This is the same structure as the integral function that yields arcsin x. If we follow the same steps as with the previous integral, then we get:
     k ∫ dx⁄√(b² − x²)   →   k ∫ dx⁄√(b²(1 − x²⁄b²)   →   k ∫ dx⁄√(b²)√(1 − (x⁄b)²)   →   k⁄b ∫ dx⁄√(1 − (x⁄b)²)    u = x⁄b, dx = b du   →   k⁄b × b ∫ du⁄√(1 − u²)   →   k arcsin u + C   →   k arcsin (x⁄b) + C
Follow these steps to integrate any function that has an inverse trig anti-derivative.

Integration by Parts

The formula for integration by parts is:
     ∫ u dv = uv − ∫ v du
If we have a function that cannot be integrated using the other integration methods, such as ∫ xln x dx, then we must use integration by parts.
For the integral I just stated, we would set u = ln x and dv = x dx. This is because it is easier to find v, which is equivalent to ∫ dv .
Since we don't know the integral of natural logarithm, we need to set dv = x dx. Now, our integral is ∫ u dv, which means it is the same as uv − ∫ v du. v = ∫ dv = ∫ x dx = ½ x2 and du = dx⁄x. If we substitute these values into our formula, we get:
      ln x × ½ x2 − ∫ ½ x2 × 1⁄x dx = ½(x2 ln x − ∫ x dx) = x2⁄2 (ln x − ½) + C

Reduce Power Method

Say we have the integral:
      ∫ sin3x cos4x dx
Using our knowledge of trigonometric identities (see Precalculus), we can simplify the integral to more manageable pieces:
      ∫ sin x sin2x cos4x dx   →   ∫ sin x cos4x (1 − cos2x) dx   →   ∫ sin x cos4x dx − ∫ sin x cos6x dx
Now our function is much simpler and can be integrated using u-substitution. This is because we have only composite function with inside function cos x. Therefore, we set u = cos x and sinx dx = -du:
      -∫ u4 du − (-∫ u6 du)   →   -1⁄5 u5 + 1⁄7 u7 + C   →   1⁄7 cos7x − 1⁄5 cos5x + C

Here's another example:
      ∫ cos3x⁄√(sin x) dx   →   ∫ cos x cos2x⁄√(sin x) dx   →   ∫ cos x(1 − sin2x)⁄sin½x dx   →   ∫ cos x⁄sin½x dx − ∫ cos x sin2x sin½x dx
Since our inside function of the composite function is sin x, u = sin x and du = cos x dx:
      ∫ u-1⁄2 du − ∫ u3⁄2 du   →   1⁄2√u − 2⁄5√u5 + C   →   1⁄2√(sin x) − 2⁄5√(sin5x) + C

What about the double-angle identities? Well, those come into play as well. Take the following integral:
      ∫ cos4x dx
To break this down into more manageable pieces, it would be better if we use the double-angle identity of cos2x. Since cos4x is the same as (cos2x)2, we can replace cos2x with ½(cos 2x + 1). Now, we simplify:
      ∫ [½(cos 2x + 1)]2 dx   →   ¼ ∫ (cos22x + 2[cos 2x] + 1) dx   →   ¼ ∫ cos22x dx + ½ ∫ cos 2x dx + ¼ ∫ dx
Now we have a problem again... cos22x is a monster function. However, we can turn this into its double angle ½(cos 4x + 1):
      ¼ ∫ ½(cos 4x + 1) dx + ¼ sin 2x + ¼ x   →   ⅛ ∫ cos 4x + ⅛ ∫ dx + ¼ sin 2x + ¼ x   →   (sin 4x)⁄32 + ⅜ x + ¼ sin 2x + C
