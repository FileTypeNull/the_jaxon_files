Derivatives

A derivative is a function that shares a tangent point to another function at a certain value of x.

Traditional definition: f '(c) = limh → 0 (f(c + h) − f(c)) ÷ h.
Alternate definition: f '(c) = limx → c (f(x + c) − f(c)) ÷ (x − c).

Basic Differentiation Using Derivatives

First Derivative: f '(x), y ', dy⁄dx
  → the derivative of a function
Second Derivative: f ''(x), y ", d²y⁄dx²
  → the derivative of the derivative of a function
Power Rule: if f(x) = xn, f '(x) = n×xn − 1
  → ex: f(x) = 3x², f '(x) = 6x

Addition Rule: if f(x) = g(x) ± h(x) ± …, f '(x) = g'(x) ± h'(x) ± …
  → ex: f(x) = x3 − 7x + 5, f '(x) = 3x² − 7

Product Rule: if f(x) = g(x) × h(x), f '(x) = g'(x)×h(x) + h'(x)×g(x)
  → ex: f(x) = (4x² + 3)√(x − 3), f '(x) = 8x√(x − 3) + ½(x − 3)-½(4x² + 3)

Quotient Rule: if f(x) = g(x) ÷ h(x), f '(x) = [g'(x) × h(x) − h'(x) × g(x)] ÷ [h(x)]²

Derivatives of Trigonometric Functions

  • d(sin x)⁄dx = cos x
  • d(cos x)⁄dx = -sin x
  • d(tan x)⁄dx = sec²x
  • d(sec x)⁄dx = tan x×sec x
  • d(csc x)⁄dx = -cot x×csc x
  • d(cot x)⁄dx = -csc²x
Chain Rule

How do you find the derivative of a function within another function, like sin(x²)? Is it just cos(x²)? No. When differentiating, it is important that the derivatives ALL functions be included in the solution.
The chain rule is defined as: d[(f ◦ g)(x)]⁄dx = (f ' ◦ g)(x) × g'(x)

If you are given the equation y = sin((3 − 7x)²), what is y '?
The best method for using chain rule is to break it down into various functions. There are three functions within the equation, so it can be rewritten as (f ◦ [g ◦ h])(x).
Next, define the functions and their derivatives:
• f(x) = sin x, f '(x) = cos x
• g(x) = x², g'(x) = 2x
• h(x) = 3 − 7x, h'(x) = -7
Finally, set up the chain rule: y ' = cos((3 − 7x)²) × d[(3 − 7x)²]⁄dx.
Because there is a composite function within the composite function, the chain rule must be applied once more. In fact, for an equation comprised of k functions, the chain rule must be applied k−1 times.
Thus, the final answer (unsimplified) is: y ' = cos((3 − 7x)²) × 2(3 − 7x) × -7.

Beware of composites! The derivatives of sin²x and sin(x²) are not the same! The result depends on the "outside" and "inside" functions!

We want to sketch the derivative graph of p'(x). Using derivative rules:
• When the function is horizontal, the derivative function is at zero
• A curved function has either a linear or curved derivative

Tangent Lines

We all know the equation of a line is y = mx + b. Well, another way to write this is y − y0 = m(x − x0), where the point ( x0, y0 ) is the point of tangency between the tangent line and original function.
m, the slope, is the value of f '(x0), where f(x) is the original function.
