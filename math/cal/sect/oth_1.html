Solving Differential Equations

Separable Differential Equations

Say we have the differential equation y' = −xy. This function can be rewritten as dy⁄dx = −xy. This is called a separable differential function because it can be separated into two different halves: a y-side and an x-side. If we divide both sides by y and then multiply both sides by dx we have: dy⁄y = −x dx. It looks like an integral on either side without the integral notation. Thus, let's add that notation and integrate:
      ∫ 1⁄y dy = − ∫ x dx  →  ln y = −½x2 + C
If needed, you can solve for y while remembering that the value of C is representative of ANY constant, so manipulating it with another constant will always yield C.

If we are given a particular solution for the equation above such as f (0) = 2, then we must solve for that particular value of C. If I plug in y = 2 and x = 0, then I get that C must be equal to 2 for this particular solution. Therefore, the specific solution for the differential equation would be:
y = f (x) = 2e−½x2

Proportionality

Introduction
Let's say that we can model the growth rate of bacteria because the rate of growth is proportional to the number of bacteria. The number of bacteria in thousands is represented using B and the time in hours is represented with t. At t = 0, there are 1,000 bacteria and at t = 5, there are 500,000 bacteria. Create a model equation, B(t), for the number of bacteria at time t.

Lesson
Okay, let's take a look at this question. It says that the rate of growth (the derivative/slope) is proportional to the number of bacteria. This calls for a proportionality constant k as we do not know the exact proportion. Now, let's write out this equation: dB⁄dt = kB. We can use separation of differential equations to solve for the general solution to B(t):
      ∫ 1⁄B dB = k ∫ dt  →  ln B = kt + C  →  B(t) = Cekt
The next step is to input the particular points into the function in order to solve for k and C in the particular solution. In this case, C = 1 and k = 1⁄5×ln(500). This makes the particular solution, when simplified:
B(t) = (500)t⁄5
The reason for this resulting in such a simple function is because we have solved for a model that represents some proportion. That is how you work with a problem dealing with proportionality.

Euler's Method

Introduction
We know that the slope-point formula is y − y0 = m(x − x0). If we think about this in terms of a change in slope, the slope m can be represented using the derivative of the function and the changes in x and y can be represented by Δ (final − initial): yf − yi = y 'i Δx. Next, let's declare the "initial" values to be relative to some number n and then solve for yf :   yn + 1 = yn + y 'n Δx.
Using this formula to approximate the next value of y when given the differential equation and a particular solution is called Euler's method.

This is an unfilled Euler's method table. If we are given a differential equation and a point on the function, we can approximate f (x) using a certain number of steps and an x-distance (Δx) between each step.
For instance, let's say that dy⁄dx = x + 2y and that f (0) = 1. Find f (1) using four equal steps:
n xn yn y 'n yn + 1 = yn + y 'n Δx

4 1 5.939 — —
Therefore, the approximate answer is that f (1) is around 5.939.
