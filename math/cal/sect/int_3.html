Improper Integrals

Introduction:
We understand what defines an integral, as it is a measure of the area underneath a function. But there are sometimes when the area is essentially infinite or unsolvable due to there being a discontinuity within the function. These are called improper integrals.
Take, for instance,   ∫1-2 1⁄x2 dx.
There is a vertical asymptote at x = 0, which is within the constraints of the definite integral. This integral is also divergent, meaning the limit of the improper integral is either infinite or does not exist (in this case, does not exist because there is a discontinuity within the constraints). In order to discern if an improper integral is divergent or convergent, meaning that the integral does have a limit that it approaches, one must evaluate it to see if it approaches a certain value.

Concept
The concept of an improper integral is easy to break down. An integral is improper if there is either a discontinuity within the bounding constraints or one of the constraints is infinity.

Example(s):
  • ∫0−∞ ex dx

  • ∫e0 ln x dx

  • ∫21 1⁄√(x − 1) dx


In order to "fix" an improper integral, we must rewrite it as a limit as it approaches the discontinuity or infinity.
For instance, ∫0−∞ ex dx should be rewritten as lima → −∞ ∫0a ex dx.
IF there are two discontinuities or both constraints are infinity, then we can rewrite it as a sum of two limits.
For instance, ∫2-1 1⁄x2 dx should be rewritten as limb → 0− ∫b-1 1⁄x2 dx + lima → 0+ ∫2a 1⁄x2 dx.

Area Between Curves

Introduction
Find the area between the functions g(x) = (x − 2)2 + 1 and f (x) = x + 1.
This can be fund by finding the difference between the area under f (x) within the points of intersection between the two functions [a, b] and the area under g(x) from a to b.
Therefore, the area is ∫ba f (x) dx − ∫ba = ∫ba [f (x) − g(x)] dx.
Now, we find the points of intersection by finding the values of x where f(x) = g(x). This makes a = 1 and b = 4. Now, we evaluate the formula we derived for the area, leading to the answer of  9⁄2.

Lesson
That was an example of finding the area between an upper and lower function. But what if the function depends on y instead of x? This would make use of methods to find the area between a left and right function.
Say we want to find the area between 4 − x = y2 and y = x − 2. It would be easier if we use y as our independent variable for both equations. This means we rewrite them as x = 4 − y2 and x = y + 2.
Now, we put everything in terms of y. We'll say the function on the right serves the same purpose as the upper function. This makes the formula for the area:
      ∫ba (4 − y2) dy − ∫ba (y + 2) dy = ∫ba (4 − y2 − y − 2) dy = ∫ba (2 − y − y2) dy
Now, we need to find the points of intersection, a and b. This is when 4 − y2 = y + 2. Thus, a = -2 and b = 1. Let's input these points into the formula and evaluate:
      ∫1-2 (2 − y − y2) dy = (2y - ½ y2 − ⅓ y3)|1-2 = 9⁄2

Finding Volume

Introduction:
Say we have a rectangle with width 3 and height 4. If we rotate it about the x-axis to form a three-dimensional shape, what volume would it fill?

The radius of this cylinder would be the height of this rectangle since the axis of rotation is the x-axis, and the rectangle's width is the cylinder's height. Let's label this rectangle as the function R(x) and say it is bounded by x = -1 and x = 2, as these will be our radial edges.
How do we find the area of this function? By using an integral, of course! To find the area, consider the following formula:
      A = ∫2-1 R(x) dx
But, how do we find the volume of the cylinder formed by the rotation of the rectangle? Well, a cylinder is just a bunch of circles stacked on top of each other. Since there can be an infinite number of circles in the cylinder, we can say that R(x) encapsulates the radii of each circle on the cylinder. How do we find the volume, though? Well, the volume is just the sum of the areas of an infinite number of circles. To find the area of a circle, we use πr2. Since R(x) is the magnitude of the radius at point x, we can use the function R to represent the radii in the equation for area:
      A = π[R(x)]2
Therefore, we can model the volume as an infinite sum–or integral–of the area of the infinite points between the radial edges:
      V = ∫2-1 π[R(x)]2 dx

Lesson
This is known as the "disk method" for finding a rotational volume, as it deals with using the area of a circle to find the volume of a cylindrical figure.

How would we find the volume if we rotate the area between two functions around an axis of rotation? We would use the "washer method" because there is empty space between the area and he axis of rotation. It is a washer because it has an inner and outer radius. You can see the washer method at work here. The main formula for the washer method is:
      V = ∫ba π[R2 − r2] dx
R is the distance between the outer function and the axis of rotation, meaning it is the outer radius. r is the distance between the inner function and the axis of rotation, meaning it is the inner radius.

The next method is known as the "cross section" method as it deals with, who would've thought, cross sections. You first take your two-dimensional area, and then you extend it into the third dimension using a known cross section, such as a square, perpendicular to one of the two axes. The area of this known cross section is labelled using the function A(x). If we take the sum of an infinite number of these cross sections across the definite span of the function from a to b. This requires an integral.
