Determining the Behavior of a Function

You can determine a function's behavior by looking at its derivative. If the value of f '(c) is positive, then f (x) is increasing at x = c. If f '(c) is negative, then f (x) is decreasing at x = c. If f '(c) is zero, then f (x) has a horizontal tangent line at x = c. If f '(c) is undefined, then f (x) has a vertical tangent line at x = c.
The critical points of a function, f (x), are the points where f '(x) = 0 or f '(x) is undefined. For instance, the critical points of y = ⅓ x3 − 4x are found using y ', which is x2 − 4, so the values are xcrit = -2, 2 because those values make y ' = 0. We can express the intervals of increase and decrease through a table:

Because the function is also the same as (x + 2)(x − 2), we can break down the function into two functions in the table. Then, we multiply the signs together to get whether the function is increasing or decreasing on that interval. This means the function is increasing on the interval(s) (-∞, -2) and (2, ∞), and is decreasing on the interval (-2, 2). Therefore, there is a relative maximum at x = -2 and a relative minimum at x = 2. Finding extrema by interpreting an interval table for the first derivative is called the first derivative test.

The second derivative can show the concavity of a function. If f ''(c) is positive at x = c, then f (x) is concave up at x = c. If f ''(c) is negative at x = c, then f (x) is concave down at x = c. If f ''(x) = 0, then that defined value of x is an inflexion point on the graph of f (x), or a point where the function changes concavity.
For the function y = ⅓ x3 − 4x, the second derivative would be y '' = 2x. This means our point of inflexion is xinflex = 0. Let's do the same thing we did with the first derivative and set up an interval table:

This means the function is concave down on the interval (-∞, 0) and concave up on the interval (0, ∞).
We can use the second derivative to find extrema as well using what is known as the second derivative test. Our first step is to take the critical points of y ', which were -2 and 2, and plug them into the second derivative. If the result is negative, the concavity at that point is down, so there is a maximum. If the result is positive, the concavity at that point is up, so there is a minimum. However, the second derivative test may fail if there are undefined points on the function.

Applications of Differentiation

Using differentiation, we can solve a lot of complex problems by interpreting them as functions. For instance, say we were given the following problem:
