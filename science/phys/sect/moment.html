Law of Conservation of Momentum

Momentum is an intertial quantity that can be measured using the following equation: p = mv
Because momentum is related to inertia, there is always a conservation of momentum when objects collide.

Inertia, in the physical sense, is the resistance to change in motion. In this case, momentum is transferred between objects in an isolated system (meaning there are no external forces). This is known as the Law of Conservation of Momentum.

This conservation can modeled using the following equation: pi = pf

Elastic Collisions


An elastic collision is a collision in which there is no net loss of kinetic energy. Thus, there is both a conservation of momentum and kinetic energy. In this case, if a ball were to collide with a surface elastically, the ball should return with the same speed but in the opposite direction. Since velocity is a vector, the momentum will have the same magnitude but opposite sign.
A new equation must be formulated for this collisions to follow the law of conservation of momentum. Assuming that there is no change in mass (collisions often result in a loss of mass as thermal energy), an elastic collision between two objects can be represented accordingly:
m1v1i + m2v2i = m1v1f + m2v2f

In other words, it is just the expanded version of Δp1 = −Δp2
